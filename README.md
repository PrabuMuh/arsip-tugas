# arsip-tugas
Proyek Sistem Arsip Tugas Mahasiswa ini dibuat sebagai media untuk membantu mahasiswa dalam mengelola dan menyimpan dokumen tugas perkuliahan secara terpusat dan terstruktur. Sistem ini memungkinkan pengguna untuk mengunggah tugas dalam format PDF, mengelompokkan arsip berdasarkan mata kuliah dan semester, serta mengunduh kembali dokumen yang telah disimpan.

Tujuan utama pembuatan proyek ini adalah untuk mengurangi risiko kehilangan file tugas, mempermudah pencarian dokumen akademik, serta memberikan pengalaman dalam pengembangan aplikasi web yang terintegrasi antara frontend, backend, database, dan cloud storage. Proyek ini juga dikembangkan sebagai bagian dari tugas mata kuliah Rekayasa Perangkat Lunak dan disesuaikan dengan ruang lingkup pengerjaan individu.
